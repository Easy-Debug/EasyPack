# EasyPack
EasyPack 是一款面向 Windows 的本地可执行文件压缩工具：拖入 .exe / .dll / .sys 即可生成 *_packed 成品，内置 PackLite（弱化常见 UPX 特征）与 UPX（体积更小）双引擎，支持批量压缩与授权管理。
